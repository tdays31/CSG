# Modern Form Designer

ID 카드 스타일의 폼을 디자인하고 이미지로 저장할 수 있는 웹 기반 폼 디자이너입니다.

## 🎨 주요 특징

- **모던한 글래스모피즘 디자인**: 부드러운 블루 계열의 그라데이션과 투명 효과
- **텍스트 중심 ID 카드 레이아웃**: 직원증/신분증 스타일의 깔끔한 디자인
- **완전한 클라이언트 사이드 처리**: 서버 저장 없이 브라우저에서 모든 작업 완료
- **이미지 내보내기**: HTML2Canvas를 활용한 고해상도 PNG 저장
- **실시간 커스터마이징**: 색상, 폰트, 그라데이션 등 다양한 디자인 요소 조정 가능

## 🚀 주요 기능

### 1. 기본 정보 입력
- **이름**: 한글/영문 이름 입력 (스택 레이아웃)
- **출생 정보**:
  - 한국식 나이 기반 출생년도 자동 계산 (2036년 기준)
  - 월/일 입력으로 상세 정보 표시
  - 만 나이 자동 계산
  - "불명" 토글 옵션
- **상세 정보**: 혈액형, 입사일, 부서
- **한마디**: 상단 개인 메시지

### 2. 성격란 시스템
- **메모 스티커 스타일**: 컨테이너 왼쪽에 70% 돌출된 라벨
- **개별 색상 커스터마이징**: 각 성격란마다 배경색/텍스트색 설정 가능
- **동적 추가/삭제**: "+ 성격 추가" 버튼으로 자유롭게 관리
- **인라인 편집**: contentEditable로 직접 텍스트 수정
- **자동 너비 동기화**: 가장 긴 텍스트에 맞춰 모든 스티커 너비 통일

### 3. 이미지 업로드
- **프로필 이미지**: 왼쪽 상단 원형 이미지
- **투명 배경 처리**: 배경 그라데이션 자동 적용
- **배경 커스터마이징**: 투명 이미지용 그라데이션 색상 조정 가능

### 4. 오너 성향 섹션
- **토글 가능**: 필요시 활성화/비활성화
- **4가지 항목**: 페티시, 킹크, 좋아요, 트리거
- **자유로운 입력**: 각 항목별 상세 설명 가능

### 5. 고급 커스터마이징
- **색상 테마**: Primary/Secondary 색상 조정
- **폰트**: Google Fonts 통합 (폰트명 입력)
- **입력 필드 스타일**:
  - 배경색/테두리색
  - Border Radius 조정
  - 그림자 강도 조절
- **텍스트 색상**: 메인/라벨 색상 개별 설정
- **그라데이션**: 배경 그라데이션 색상 조정

### 6. 이미지 저장
- **고해상도 출력**: Scale 2배 적용 (고품질 PNG)
- **최소 너비 보장**: 1200px 최소 너비로 저장
- **불필요 요소 제외**: 저장 버튼과 +성격 추가 버튼 자동 제외
- **투명 배경**: 배경 투명 옵션 지원

## 🛠️ 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**:
  - CSS Custom Properties (CSS 변수)
  - Flexbox & Grid Layout
  - Glassmorphism 효과
  - 그라데이션 배경
- **Vanilla JavaScript**:
  - DOM 조작
  - 이벤트 핸들링
  - 동적 스타일 변경
- **HTML2Canvas**: 이미지 캡처 라이브러리
- **GitHub Pages**: 정적 사이트 호스팅

## 📦 배포

이 프로젝트는 GitHub Pages로 배포되도록 설계되었습니다.

```bash
# main 브랜치에 index.html 포함 후
# GitHub Repository Settings > Pages 에서 배포 설정
```

## 💡 사용 방법

1. **정보 입력**: 각 필드에 정보 입력
2. **커스터마이징**: 우측 상단 "🎨 디자인 커스터마이징" 버튼 클릭
3. **성격란 추가**: "+ 성격 추가" 버튼으로 성격란 생성 및 편집
4. **이미지 업로드**: 프로필 이미지 업로드
5. **저장**: "💾 이미지로 저장" 버튼 클릭하여 PNG 다운로드

## 🎯 디자인 특징

### 레이아웃
- **고정 너비**: 1200px 고정 (반응형 제거, 웹 전용)
- **ID 카드 스타일**: 텍스트 중심, 투명 배경
- **스택형 입력**: 한글/영문 이름, 출생 정보 등 수직 배치
- **정렬 최적화**:
  - 한글 이름 라인 ↔ 출생년월일 라인 (30px)
  - 영문 이름 라인 ↔ 혈액형/입사일/부서 라인 (24px)

### 색상 시스템
- **기본 Primary**: #667eea (소프트 블루)
- **기본 Secondary**: #764ba2 (퍼플)
- **텍스트**: #333333 (다크 그레이)
- **라벨**: #999999 (그레이)
- **테두리**: 투명, 호버 시 점선 표시

### 상호작용
- **호버 효과**: 입력 필드 호버 시 점선 테두리 표시
- **포커스**: 부드러운 아웃라인 효과
- **버튼**: Transform과 그림자로 입체감
- **패널**: 슬라이드 인/아웃 애니메이션

## 📝 개발 히스토리

### 초기 설계
- 글래스모피즘 기반 모던 디자인 구축
- 기본 폼 레이아웃 및 입력 필드 구성

### 디자인 개선
- ID 카드 스타일로 전환 (테두리 제거, 텍스트 강조)
- 점선 호버 효과 적용
- 색상 톤 조정 (#000 → #333)

### 기능 추가
- 성격란 시스템 구현 (메모 스티커 스타일)
- 개별 색상 커스터마이징 기능
- 한국식 나이 계산 시스템 (2036년 기준)
- 출생 불명 토글 기능
- 이미지 업로드 및 배경 처리

### 레이아웃 최적화
- 혈액형/입사일/부서 필드를 출생 섹션 하단으로 이동
- 라벨 없는 작은 필드 (12px) 디자인
- 시각적 정렬 (이름 라인 ↔ 출생/상세 라인)
- 고정 높이 설정으로 안정적인 디자인

### 저장 기능
- HTML2Canvas 통합
- 1200px 최소 너비 보장
- 버튼 요소 자동 제외 (no-export 클래스)
- 텍스트 크롭 방지 (line-height 최적화)

### 반응형 제거
- 1200px 고정 너비로 변경
- 웹 전용 디자인으로 전환
- 작은 화면에서 스크롤 지원

## 🐛 해결된 이슈

1. **접근성 경고**: label과 input 연결 (`for`, `aria-label`)
2. **스티커 X 버튼 삭제**: `contentEditable="false"` 설정
3. **영문 이름 색상**: `!important`로 그레이 색상 고정
4. **GitHub Pages 404**: main 브랜치에 index.html 병합
5. **불명 토글 버그**: 해제 시 `calculateBirthInfo()` 재호출
6. **저장 시 버튼 포함**: `ignoreElements` 옵션 추가
7. **텍스트 크롭**: `line-height` 추가로 렌더링 개선
